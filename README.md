# D20-dice-for-C1
在自由象限C1显示屏上运行的二十面骰子程序。

第一次打开时会进行校准，只用一只眼睛正对小屏幕，鼠标移动使左右两侧的绿条与中间的白色处于同一水平线，点击鼠标完成校准。后续想重新校准可以按P键。如果是黑屏，请先启动OpenstageAI。

鼠标拖动投出骰子，也可以单击鼠标左键或空格进行随机投掷。鼠标滚轮可以放大缩小，WS控制入屏/出屏效果，R还原，ESC或在底部任务栏退出，静止状态下方向键可以翻转骰子。

想要更换背景，可以进入文件夹，修改bg_rgbd.bmp，这是用来生成背景的深度图。

想要修改皮肤，可以进入dice_surface文件夹下，dice.bmp是展开图，如果要修改它请确定三角形的排布和文字的朝向与示例的一致。01-20.bmp是单独的各个面，要修改的话需要让三角形底边的两个角在底边的角落，顶点在顶边的中点。如果dice.bmp存在则优先使用它，不想用的话就改个名字。皮肤需要手动替换。
